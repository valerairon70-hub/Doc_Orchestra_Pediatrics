  КАК ПРОДОЛЖИТЬ РАБОТУ С DOC ORCHESTRA
  (читай это перед каждой новой сессией в VS Code)
========================================================


ШАГ 1 — ОТКРЫТЬ ПРОЕКТ
-----------------------
Открыть VS Code → File → Open Folder →
Выбрать папку: Doc_Orchestra_Pediatrics


ШАГ 2 — ОТКРЫТЬ ТЕРМИНАЛ
--------------------------
В VS Code: Terminal → New Terminal
(или сочетание клавиш: Ctrl+` )


ШАГ 3 — ЗАПУСТИТЬ СЕРВЕР
--------------------------
Вставить в терминал и нажать Enter:

    .venv/bin/python3 mvp.py

Ты увидишь:
    ✅ REAL MODE — Claude API подключён.
    Открой в браузере:
    📱 Родитель  →  http://localhost:8081/parent
    🩺 Врач      →  http://localhost:8081/cockpit

Если появилась ошибка "address already in use" — порт занят старым
процессом. Выполни эту команду, затем снова запусти сервер:

    lsof -ti :8081 | xargs kill -9


ШАГ 4 — ОТКРЫТЬ В БРАУЗЕРЕ
----------------------------
Родительский чат:  http://localhost:8081/parent
Кокпит врача:      http://localhost:8081/cockpit

Пароль кокпита: смотри в файле .env строку COCKPIT_PASSWORD
(по умолчанию: doctor123)


ШАГ 5 — НАПИСАТЬ МНЕ (Claude) И ПРОДОЛЖИТЬ РАБОТУ
---------------------------------------------------
В VS Code открыть Claude (иконка в боковой панели).
Написать: "Продолжаем работу"

Я автоматически прочитаю файл CLAUDE.md и сразу буду знать:
- на какой фазе мы остановились
- что уже сделано
- что делать дальше
- все правила проекта

Не нужно объяснять контекст заново — я всё вспомню сам.
